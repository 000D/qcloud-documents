## 1. 接口描述
本接口(CertGetList)用于获取证书列表。
接口请求域名：<font style="color:red">wss.api.qcloud.com</font>

## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见<a href="/doc/api/372/4153" title="公共请求参数">公共请求参数</a>页面。其中，此接口的Action字段为CertGetList。  

| 参数名称 | 是否必选  | 类型 | 默认值 | 描述 |
|---------|---------|---------|---------|---------|
| page | 否 | String | 要操作的域名（主域名，不包括 www，例如：qcloud.com） |
| count | 否 | Int | 解析记录的ID
| searchKey | 否 | String | 子域名，例如：www |
| certType | 否 | String | 记录类型，可选的记录类型为："A", "CNAME", "MX", "TXT", "NS", "AAAA", "SRV"
| id | 否 | String | 记录的线路名称，如："默认" |
| withCert | 否 | String | 记录值, 如 IP:192.168.10.2, CNAME: cname.dnspod.com., MX: mail.dnspod.com.
| altDomain | 否 | Int | TTL 值，范围1-604800，不同等级域名最小值不同，默认为 600
