## 1. 接口描述
天御反欺诈服务，主要应用于银行、证券、保险、P2P等金融行业客户，通过腾讯的大数据风控能力，可以准确识别恶意用户信息，解决客户在支付、活动、理财，风控等业务环节遇到的欺诈威胁，降低企业的损失。

协议：HTTPS

域名：csec.api.qcloud.com

接口名：AntiFraud

## 2. 输入参数
