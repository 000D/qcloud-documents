## 1. API Description
Domain: live.api.qcloud.com
API name:  DescribeLVBOnlineUsers

Note:  Measure the number of online users based on the number of access connections to a video URL. If the RTMP downstream protocol is used, the number of online users can be obtained based on the analysis data. If the HLS or HLS/RTMP downstream protocol is used, since HLS implements access based on the TS fragment, the real-time access connections may not accurately reflect the number of online users.
