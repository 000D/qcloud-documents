## 1. API Description

Domain: live.api.qcloud.com
API name:  CreateLVBShot

Note: Screenshot files are stored on the COS platform, for long-term storage. Before using this function, you need to activate relevant services. Related costs (including storage, and so on) are charged by using the COS billing mode. See "References" (http://www.qcloud.com/product/cos.html).

Note:
1. Screenshots are based on the original bit rate intercept and the frequency is one screenshot every 10 seconds, which cannot be modified.
2. The output format is JPG.
3. You are expected to get the screenshot file in 20 seconds minutes after the start of the screenshot task.
