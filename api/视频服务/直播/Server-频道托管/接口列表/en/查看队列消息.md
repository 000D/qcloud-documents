## 1. API Description

Domain: live.api.qcloud.com
API name:  DescribeQueueLog

Note: This API is used to query the queue of a screenshot and can be used for other message queue services.
The screenshot file is stored in the COS which provides long-term storage function. Before using this function, you need to activate relevant services. Related costs (including storage, and so on) are charged by using the COS billing mode. See "References" (http://www.qcloud.com/product/cos.html).

**Note: This service must be provisioned for the queue query API**. You can submit an application through the after-sales QQ number 514025596 to provision the service. Indicate the message "Want to activate the LVB screenshot query queue API service" and the account information in the application. The service will be activated in one working day. You can also choose to perform query through COS relevant APIs. In this case, you do not need to provision this service.
