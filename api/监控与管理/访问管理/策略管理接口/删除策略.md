## 接口描述

本接口（DeletePolicy ）可用于删除策略。
请求域名：

```
cam.api.qcloud.com 
```

##  输入参数

以下请求参数列表仅列出了接口请求参数，未列出 [公共请求参数](https://cloud.tencent.com/document/api/213/6976)。

| 参数名称 | 类型  | 必选 | 描述                                      |
| -------- | ----- | ---- | ----------------------------------------- |
| policyId | array | 是   | 数组，数组成员是策略 id，支持批量删除策略 |

## 输出参数

| 参数名称 | 类型 | 描述 |
| -------- | ---- | ---- |
|          |      |      |
