## 接口描述

本接口（DetachUserPolicy）可用于解除绑定策略到用户。

请求域名：

```
cam.api.qcloud.com 
```

##  输入参数

以下请求参数列表仅列出了接口请求参数，未列出 [公共请求参数](https://cloud.tencent.com/document/api/213/6976)。

| 参数名称 | 类型 | 必选 | 描述       |
| -------- | ---- | ---- | ---------- |
| policyId | int  | 是   | 策略 id    |
| uin      | int  | 是   | 子账号 uin |

## 输出参数

| 参数名称 | 类型 | 描述 |
| -------- | ---- | ---- |
|          |      |      |
