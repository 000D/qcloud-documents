## 1. 接口描述
接口 ( DescribeClusterTaskResult )查询异步任务结果。
接口请求域名：<font style="color:red">ccs.api.qcloud.com</font>

## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，其它参数见 [公共请求参数](/doc/api/457/9463) 页面。
| 参数名称 | 是否必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| requestId   | 是    | Int | 异步任务 ID |

## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码。0 表示成功，其他值表示失败|
| codeDesc | String |业务侧错误码。成功时返回 Success，错误时返回具体业务错误原因。|
| message | String | 模块错误信息描述，与接口相关。详见错误码页面的模块错误码。|
| status | String | succ，fail，running。|

## 4. 示例
输入
```
  https://domain/v2/index.php?Action=DescribeClusterTaskResult
  &requestId=123
  &其它公共参数
```
输出
```
{
    "code": 0,
    "message": "",
    "codeDesc": "Success",
    "data": {
        "status": "succ"
    }
}
```
