## 1. 接口描述
 
本接口 (CreateClusterNamespace) 用于创建命名空间。

接口请求域名：<font style="color:red">ccs.api.qcloud.com</font>



## 2. 输入参数

以下请求参数列表仅列出了接口请求参数，其它参数见[公共请求参数](https://www.qcloud.com/document/api/214/4183)页面。

| 参数名称 | 是否必选  | 类型 | 描述 |
|---------|---------|---------|---------
| clusterId   | 是    | String | |集群ID，可通过查询集群接口反回字段中的 clusterId获取。 |
| name   | 是 | String       |命名空间名称|


## 3. 输出参数
 
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码。0表示成功，其他值表示失败|
| message | String | 模块错误信息描述，与接口相关|


## 4. 示例

输入

```
  https://domain/v2/index.php?Action=CreateClusterNamespace
  &clusterId=cls-xxxxx
  &name=tst
  &其它公共参数
```

输出

```
  {
    "code": 0,
    "message": "", 
}

```
