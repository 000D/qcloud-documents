## 1. 接口描述
本接口（ModifyTdsqlInstanceConfig）用于修改数据库参数。
接口请求域名：**_tdsql.api.qcloud.com_**

## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见[公共请求参数](/doc/api/309/7016)页面。
其中，此接口的 Action 字段为 ModifyTdsqlInstanceConfig。

| 参数名称| 类型| 是否必选| 描述|
|---------|---------|---------|---------|
| Version|String|是|表示 API 版本号，主要用于标识请求的不同 API 版本。 本接口第一版本可传：2017-03-12。|
| CdbInstanceId | String | 是 | 实例 ID|
| Config.N | [TdsqlParam objects](/doc/api/213/9451#tdsqlparam) | 是 | 参数列表。每一个元素是Param和Value的组合,见示例|

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId| String| 唯一请求 ID。每次请求都会返回 `RequestId`。当用户调用接口失败找后台研发人员处理时需提供该 `RequestId`。|
| CdbInstanceId | String | 实例 ID |
| Config | [Array of TdsqlConfigResult objects](/doc/api/213/9451#tdsqlconfigresult)| 各参数设置结果列表，见示例。|

## 4. 错误码表

以下是本接口常见的错误码，如果有不在此列的错误请查阅[TDSQL错误码表](/doc/api/309/7150)

| 错误码 | 描述 |
|---------|---------|
| DbOperationFailed | DB 内部失败 |
| InstanceAlreadyDeleted | 实例已删除 |
| InstanceStatusAbnormal | 实例状态异常（非删除），不能进行操作 |
| OssOperationFailed | OSS 内部失败 |
| InstanceHasBeenLocked | tdsql 已经被其它流程锁定 |
| GetDbconfigFailed | 获取参数失败 |

&<<a href="https://cloud.tencent.com/doc/api/229/6976">公共请求参数</a>>

&CdbInstanceId=tdsql-fdpjf5zh
