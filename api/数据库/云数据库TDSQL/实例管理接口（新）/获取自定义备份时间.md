## 1. 接口描述
本接口（DescribeTdsqlBackupTime）用于获取自定义备份时间。
接口请求域名：**_tdsql.api.qcloud.com_**

## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见[公共请求参数](/doc/api/309/7016)页面。其中，此接口的 Action 字段为 DescribeTdsqlBackupTime。

| 参数名称| 类型| 是否必选| 描述|
|---------|---------|---------|---------|
| Version|String|是|表示 API 版本号，主要用于标识请求的不同 API 版本。 本接口第一版本可传：2017-03-12。|
| CdbInstanceIds.N| array of Strings | 否 |按照一个或者多个实例 ID 查询。实例 ID 形如：`tdsql-ow728lmc`。此参数的具体格式可参考 API [简介](/doc/api/237/2246)的 `Id.N` 小节）。每次请求的实例的上限为100。|

## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId| String| 唯一请求 ID。每次请求都会返回 `RequestId`。当用户调用接口失败找后台研发人员处理时需提供该 `RequestId`。|
| Instances | [Array of BackupTimeConfig objects](/doc/api/213/9451#backuptimeconfig) | 实例备份时间配置信息| 

## 4. 错误码表

以下是本接口常见的错误码，如果有不在此列的错误请查阅 [TDSQL 错误码表](/doc/api/309/7150)

| 错误码 | 描述 |
|---------|---------|
| DbOperationFailed | DB内部失败 |
| OssOpertaionFailed | OSS内部失败 |
| OSSGetInstanceError | OSS获取实例信息错误 |
| InnerSystemError | 内部系统错误，和业务无关 |
