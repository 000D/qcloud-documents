## 1. 接口描述
注：本接口为改版后的API接口。如需了解旧接口相关信息，请参考：[查看实例列表](/doc/api/309/5447)

本接口（DescribeTdsqlInstances）用于查看实例列表。
接口请求域名：**_tdsql.api.qcloud.com_**
* 可以根据实例 ID、实例名称或者实例 VPC 等信息来查询实例的详细信息。
* 如果参数为空，返回当前用户一定数量（`Limit` 所指定的数量，默认为 20）的实例

## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见[公共请求参数](/doc/api/309/7016)页面。其中，此接口的 Action 字段为 DescribeTdsqlInstances。

| 参数名称| 类型| 是否必选| 描述|
|---------|---------|---------|---------|
| Version|String|是|表示 API 版本号，主要用于标识请求的不同 API 版本。 本接口第一版本可传：2017-03-12。|
| CdbInstanceIds.N| array of Strings | 否 |按照一个或者多个实例 ID 查询。实例 ID 形如：`tdsql-ow728lmc`。此参数的具体格式可参考 API [简介](/doc/api/237/2246)的 `Id.N` 小节）。每次请求的实例的上限为100。|
| OriginSerialIds.N| array of String | 否 |按照一个或者多个 OriginSerialId 查询。|
| Offset| Integer| 否| 偏移量，默认为 0。关于 `Offset` 的更进一步介绍请参考 API [简介](/doc/api/237/2246#.E8.BE.93.E5.85.A5.E5.8F.82.E6.95.B0.E4.B8.8E.E8.BF.94.E5.9B.9E.E5.8F.82.E6.95.B0.E9.87.8A.E4.B9.89)中的相关小节。|
| Limit| Integer| 否| 返回数量，默认为 20，最大值为 100。关于 `Limit` 的更进一步介绍请参考 API [简介](/doc/api/237/2246#.E8.BE.93.E5.85.A5.E5.8F.82.E6.95.B0.E4.B8.8E.E8.BF.94.E5.9B.9E.E5.8F.82.E6.95.B0.E9.87.8A.E4.B9.89)中的相关小节。|
| OrderBy | String | 否 | 排序字段， projectId， createtime， instancename 三者之一 |
| OrderByType | String | 否 |排序类型， desc 或者 asc|
| SearchKey | String | 否 | 实例名按此模糊搜索|
| ProjectIds.N | array of String | 否 | 按项目 ID 查询|
| IsFilterVpc | Integer | 否 | 0 为 false， 1 为 true，表示是否根据 VPC 网络来搜索|
| VpcId | Integer | 否 |私有网络 ID， IsFilterVpc 为 true 时有效|
| SubnetId | Integer | 否 | 子网的 ID， IsFilterVpc 为 true 时有效|

## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId| String| 唯一请求 ID。每次请求都会返回 `RequestId`。当用户调用接口失败找后台研发人员处理时需提供该 `RequestId`。|
| TotalCount| Integer| 符合条件的实例数量。|
| Instances| [Array of Instance objects](/doc/api/213/9451#instance)| 实例详细信息列表。|
## 4. 错误码表

以下是本接口常见的错误码，如果有不在此列的错误请查阅 [TDSQL 错误码表](/doc/api/309/7150)

| 错误码 | 描述 |
|---------|---------|
| DbOperationFailed | DB 内部失败 |
| SpecIdIllegal | 无法找到实例的规格信息 |
| FenceError | 围拢接口错误 |
| IllegalExclusterID | 非法的独享集群 ID |
