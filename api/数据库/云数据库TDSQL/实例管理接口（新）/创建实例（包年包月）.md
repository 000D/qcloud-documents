## 1. 接口描述
本接口（CreateTdsqlInstance）用于创建 Tdsql 实例。
接口请求域名：**_tdsql.api.qcloud.com_**

## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见[公共请求参数](/doc/api/309/7016)页面。
其中，此接口的 Action 字段为 CreateTdsqlInstance。

| 参数名称| 类型| 是否必选| 描述|
|---------|---------|---------|---------|
| Version|String|是|表示 API 版本号，主要用于标识请求的不同 API 版本。 本接口第一版本可传：2017-03-12。|
| Zone | String|是|实例所属的可用区ID。|
| CostType | Integer | 否 | 计费类型：0 包年包月，1 按量计费。目前仅支持包年包月。|
| Pid | Integer|是|产品类型 Id，通过[查询实例规格](/doc/api/309/5537)获取。|
| GoodsNum|Integer|是|商品数，不传则默认为 1|
| Period|Integer|否|时长，单位：月。查询包年包月类型时有效。|
| Memory|Integer|是|内存大小，单位：MB|
| Storage|Integer|是|存储空间大小，单位：MB|
| ProjectId|Integer|是|存储空间大小，单位：MB|
| VpcId |Integer|否| 私有网络 ID。|
| SubnetId |Integer|否| 私有网络子网 ID。|
| ProjectId |Integer|否| 项目 ID，可以通过[查看项目列表](/doc/api/309/5604)获取，不传默认|
| DbVersionId |String|否|数据库引擎版本，当前可选：10.0.10，10.1.9|

## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId| String| 唯一请求 ID。每次请求都会返回 `RequestId`。当用户调用接口失败找后台研发人员处理时需提供该 `RequestId`。|
| DealNames| Array of strings|订单号列表|
| CdbInstanceIds| Object |订单对应的实例 ID 列表，订单号为 Key，实例 ID 列表为值。如果此处没有返回实例 ID，可以通过订单查询接口获取。|

## 4. 错误码表

以下是本接口常见的错误码，如果有不在此列的错误请查阅[TDSQL错误码表](/doc/api/309/7150)

| 错误码 | 描述 |
|---------|---------|
|GetVpcFailed|获取 VPC 信息失败|
|InvalidParameter.VpcNotFound|未找到指定的 VPC 信息|
|GetSubnetFailed|获取 VPC 子网信息失败|
|InvalidParameter.SubnetNotFound|未找到指定的 VPC 子网信息|
|CreateOrderFailed|创建订单失败|
|PayFailed|支付订单失败|
