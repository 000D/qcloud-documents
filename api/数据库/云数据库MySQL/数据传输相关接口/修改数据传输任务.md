## 1. 接口描述
本接口(ModifyCdbDataMigrationTask)用于修改数据传输任务。
接口请求域名：<font style='color:red'>cdb.api.qcloud.com </font>


## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见<a href='/doc/api/372/4153' title='公共请求参数'>公共请求参数</a>页面。其中，此接口的Action字段为ModifyCdbDataMigrationTask。

| 参数名称 | 是否必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| jobName | 是 | String | 任务名称 |
| srcType | 是 | Int | 迁移类型（1：自建迁移。 2：外网迁移。 10：灾备同步）|
| srcIp | 否 | String | 源MySQL的ip（自建迁移 or 外网迁移 时必传）|
| srcPort | 否 | Int | 源MySQL的port（自建迁移 or 外网迁移 时必传）|
| srcUser | 否 | String | 源MySQL的用户名（自建迁移 or 外网迁移 时必传）|
| srcPw | 否 | String | 源MySQL的密码（自建迁移 or 外网迁移 时必传）|
| cvmId | 否 | String | 源MySQL的宿主cvm 实例Id（自建迁移时必传）|
| srcId | 否 | String | 源实例ID（灾备同步时必传）|
| srcRegion | 否 | String | 源实例地域（灾备同步时必传）|
| dstId | 是 | String | 目标实例ID |
| dstRegion | 是 | String | 目标实例地域 |
| expectTime | 否 | String | 任务开始时间 |
| objType | 是 | String | 对象类型，'instance'：整个实例, 'db'：指定库表 |
| migrateObject.n | 否 | String | 可选参数，类型为db时，选中的库表详情 |
| jobType | 是 | Int | 0马上执行，1定时执行 |
| jobId | 是 | Int | 任务ID |


## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码, 0表示成功，其他值表示失败。详见错误码页面的<a href='https://www.qcloud.com/doc/api/372/%E9%94%99%E8%AF%AF%E7%A0%81#1.E3.80.81.E5.85.AC.E5.85.B1.E9.94.99.E8.AF.AF.E7.A0.81' title='公共错误码'>公共错误码</a>。|
| message | String | 模块错误信息描述，与接口相关。|
| codeDesc | String | 描述（待补充） |
| data | Array | 描述（待补充） |
| data.taskId | Int | 描述（待补充）| 


## 4. 示例
输入
```
https://cdb.api.qcloud.com/v2/index.php?Action=ModifyCdbDataMigrationTask
&<公共请求参数>
&jobName=newJobName
&srcType=2
&srcIp=119.29.178.214
&srcPort=3307
&srcUser=root
&srcPw=isd@cloud
&dstId=cdb-rharhmka
&dstRegion=gz
&expectTime=2016-12-08 21:56:35
&objType=db
&migrateObject.0.database=test
&jobType=1
&jobId=206
```
输出
```
{
    "code":"0",
    "message":"",
    "codeDesc":"Success",
    "data":{
        "taskId":"206"
    }
}
```
