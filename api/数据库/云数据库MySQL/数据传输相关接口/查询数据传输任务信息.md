## 1. 接口描述
本接口(GetCdbDataTransmissionTaskInfo)用于查询数据传输任务信息。
接口请求域名：<font style='color:red'>cdb.api.qcloud.com </font>



## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见<a href='/doc/api/372/4153' title='公共请求参数'>公共请求参数</a>页面。其中，此接口的Action字段为GetCdbDataTransmissionTaskInfo。

| 参数名称 | 是否必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| taskId | 是 | Int | 数据传输任务ID，请使用[DescribeProject](/doc/api/229/1330)接口查询任务ID |


## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码, 0表示成功，其他值表示失败。详见错误码页面的<a href='https://www.qcloud.com/doc/api/372/%E9%94%99%E8%AF%AF%E7%A0%81#1.E3.80.81.E5.85.AC.E5.85.B1.E9.94.99.E8.AF.AF.E7.A0.81' title='公共错误码'>公共错误码</a>。|
| message | String | 模块错误信息描述，与接口相关 |
| codeDesc | String | 英文错误描述 |
| data | Array | 返回的数据 |
| data.taskId | String | 数据传输任务 ID| 
| data.taskName | String | 数据传输任务名称 | 
| data.appId | String | 用户的 AppId | 
| data.cvmId | String | CVM 实例 ID | 
| data.srcip | String | 迁移源 IP | 
| data.srcport | String | 迁移源 PORT | 
| data.status | String | 迁移状态，可能的返回值：<br>0-初始化<br>1-准备运行中<br>2-运行中<br>3-运行成功<br>4-运行失败<br>5-任务强行中止<br>6-被清理掉<br>提交完成请求 | 
| data.nextStatus | String | 正在进行的状态，可能的取值同data.tasks.status | 
| data.nextMsg | String | 正在进行状态对应的消息| 
| data.progress | String | 当前进度的百分比，取值范围为[0-100] | 
| data.message | String |  任务的错误信息 | 
| data.checkWorkId | String | 校验任务 ID | 
| data.checkFlag | String | 校验标签，0-未检验，1-校验通过，2-校验未通过 | 
| data.stepAll | String | 总共的步骤数量 | 
| data.stepNow | String | 当前进行的步骤 | 
| data.taskType | String | 任务类型，0-立刻执行，1-定时执行 | 
| data.migrateOption | String | 迁移类型，0-迁移结构和数据，1-只迁移结构 | 
| data.startTime | String | 任务开始执行时间 | 
| data.endTime | String | 任务结束执行时间 | 
| data.updateTime | String | 任务上一次更新状态的时间 | 
| data.submitTime | String | 任务提交的时间 | 
| data.uInstanceId | String | 源实例类型：1-CVM实例，2-外网实例，3-VPC实例，10-灾备实例 | 
| data.srcType | String |迁移源 ID | 
| data.srcId | String | 迁移目的地 CDB 实例 ID |
