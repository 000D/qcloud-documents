## 1. 接口描述
本接口(CreateCdbDataMigrationCheckTask)用于创建数据传输校验任务。
接口请求域名：<font style='color:red'>cdb.api.qcloud.com </font>



## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见<a href='/doc/api/372/4153' title='公共请求参数'>公共请求参数</a>页面。其中，此接口的Action字段为CreateCdbDataMigrationCheckTask。

| 参数名称 | 是否必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| jobId | 是 | Int | 任务ID|
| srcType | 是 | Int | 迁移类型（1：自建迁移。 2：外网迁移。 10：灾备同步）|


## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码, 0表示成功，其他值表示失败。详见错误码页面的<a href='https://www.qcloud.com/doc/api/372/%E9%94%99%E8%AF%AF%E7%A0%81#1.E3.80.81.E5.85.AC.E5.85.B1.E9.94.99.E8.AF.AF.E7.A0.81' title='公共错误码'>公共错误码</a>。|
| message | String | 模块错误信息描述，与接口相关。|
| codeDesc | String | 描述（待补充） |
| data | Array | 描述（待补充） |
| data.workid | String | 描述（待补充）| 


## 4. 示例
输入
```
https://cdb.api.qcloud.com/v2/index.php?Action=CreateCdbDataMigrationCheckTask
&<公共请求参数>
&jobId=206
&srcType=2
```
输出
```
{
    "code":"0",
    "message":"",
    "codeDesc":"Success"
}
```
