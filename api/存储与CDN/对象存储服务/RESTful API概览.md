## 描述

对象存储服务的 RESTful API 是一种轻量级的，无连接状态的，可以直接通过 http/https 的请求和响应，来和腾讯云对象存储（COS）后台进行交互操作的 Web 服务。

在查阅其他的 API 文档之前，首先请详细阅读 调用方式 和 签名鉴权 。
