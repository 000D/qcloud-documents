# 删除目录 API 
## 功能描述
使用该 API 对腾讯云对象存储中某个目录进行删除。删除目录前必须先清空目录下的文件。

>**注意**
>目标目录不可填写『/』，无法删除根目录，即无法删除 Bucket。


## 请求
语法示例：
```
POST /files/v2/<appid>/<bucket_name>/<dir_name>/  HTTP/1.1
Host: gz.file.myqcloud.com
Content-Type: application/json
Authorization: AuthString
```

### 请求参数
该请求不带请求参数。

### 请求体
该 API 接口请求的请求体具体节点内容为：
```
{
    "op": "delete"
}
```
具体内容描述如下：

|参数名称|	描述	|类型|	必选|
|---|---|---|---|
|op|	操作类型，填“delete”	|String	|是|

## 响应

### 响应体

该响应体返回为 **application/json** 数据，包含完整节点数据的内容展示如下：
```
 {
    "code": 0, 
    "message": "SUCCESS"
}
```
具体的参数描述如下：

|参数名称|描述|类型|必选|
|---|-- |--|--|
| code | 服务端返回码 | Int |  是 |
| message | 服务端提示内容 | String |  是 |
