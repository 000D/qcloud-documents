## 何谓移动直播？
移动直播并不是一个具体的服务，而是一套**帮助您在智能手机上做直播**的解决方案，它是腾讯云直播服务（LVB）在移动终端方向的技术延伸，您可以选择：

- **如果您有成熟终端解决方案**
如果您已经有成熟的终端解决方案，只是想要选择更加优秀的直播云服务，推荐您直接转 [直播云服务](https://www.qcloud.com/product/LVB) 了解云端对接解决方案即可。

- **如果您只想对接推流和播放**
如果您想要在自己的 APP 里集成直播推流和播放功能，推荐您选择 [快速集成方案](#.E5.BF.AB.E9.80.9F.E9.9B.86.E6.88.90.E6.96.B9.E6.A1.88)，接入非常简单，最快一天内就能搞定直播功能。

- **如果您想做完善的直播服务**
如果您想要在自己的 APP 里集成整套完善的直播能力，诸如文字互动、弹幕消息、飘星点赞、美颜增白、动效蒙皮、连麦互动、身份认证等功能，推荐您选择 [一体化集成方案](#.E4.B8.80.E4.BD.93.E5.8C.96.E9.9B.86.E6.88.90.E6.96.B9.E6.A1.88)。

## 快速集成方案
如果您想要在自己的 APP 里集成直播推流和播放功能，下面三步就能达成目标：
![](//mc.qcloudimg.com/static/img/92a464fc26bbe52fea8816f8e6061ef3/image.jpg)

- **Step1：开通直播服务(LVB)**
[开通](https://console.qcloud.com/live)直播服务后（需将账号通过腾讯云企业或个人认证）， 即可开始调试体验直播功能，首次开通可获得10G免费流量。

- **Step2：集成 RTMP SDK**
安排您的终端研发工程师 [下载](https://www.qcloud.com/document/product/454/7873) RTMP SDK 开发包，并参考对接文档完成接入。
  + [iOS 推流](https://www.qcloud.com/document/product/454/7879)
  + [iOS 播放](https://www.qcloud.com/document/product/454/7880)
  + [Android 推流](https://www.qcloud.com/document/product/454/7885)
  + [Android 播放](https://www.qcloud.com/document/product/454/7886)

- **Step3：对接直播流管理**
安排您的后台研发工程师，了解腾讯云服务端 API ，并进行服务端的接入。
  + [如何获取推流URL？](https://www.qcloud.com/document/product/454/7915)
  + [如何构建房间列表？](https://www.qcloud.com/document/product/454/7916)
  + [如何管理直播流？](https://www.qcloud.com/document/product/454/7920)
