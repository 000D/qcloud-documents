## 接口名称
DescribeVodHosts

## 功能说明

用于查询所有域名的信息。

*本接口只支持点播4.0*

## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟

### 参数说明
| 参数名称      | 必填 | 类型    | 说明                                                                            |
| ------------- | ---- | ------- | ------------------------------------------------------------------------------- |
| offset        | 否   | Integer | 偏移量，默认为0                                                                 |
| limit         | 否   | Integer | 指定查询的返回数量，默认全部返回                                                |
| COMMON_PARAMS | 是   |         | 参见[公共参数](/document/product/266/7782#.E5.85.AC.E5.85.B1.E5.8F.82.E6.95.B0) |

### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=DescribeVodHosts
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称 | 类型    | 说明                          |
| -------- | ------- | ----------------------------- |
| code     | Integer | 错误码, 0: 成功, 其他值: 失败 |
| message  | String  | 错误信息                      |
| data     | Object  | 结果数据,见Data说明           |

#### Data 参数说明
| 参数名称 | 类型    | 说明                         |
| -------- | ------- | ---------------------------- |
| total    | Integer | 域名总数                     |
| hosts    | Array   | 域名详细配置信息，见Host说明 |


#### Host域名详细配置
| 参数名称           | 类型    | 说明                                                                                                                   |
| ------------------ | ------- | ---------------------------------------------------------------------------------------------------------------------- |
| host               | String  | 域名                                                                                                                   |
| cname              | String  | 点播CDN 分配的 .cdn.dnsv1.com 后缀加速域名                                                                             |             |
| oversea            | Integer | 是否开通了海外加速，1：是，0：否                                                                                       |
| status             | Integer | 域名加速状态，1：域名审核中；2：域名审核未通过；3：域名审核通过处于部署中；4：域名部署中；5：域名已启动；6：域名已关闭 |
| --refer            | Object  | Refer防盗链设置，详细说明见下文                                                                                        |
| --safetyChainRefer | Object  | Key防盗链设置，详细说明见下文                                                                                          |
| sslType            | Integer | 是否开通HTTPS，0：未开通HTTPS配置；若为其他，则表示已开通HTTPS配置                                                     |
| sslDeployTime      | String  | SSL部署时间                                                                                                            |
| sslExpireTtime     | String  | SSL过期时间                                                                                                            |

#### Refer防盗链参数
| 参数名称 | 类型    | 说明                                              |
| -------- | ------- | ------------------------------------------------- |
| type     | Integer | 防盗链类型，0：未配置防盗链；1：黑名单；2：白名单 |
| nullFlag | Integer | 防盗链是否为空，若为1，则表示防盗链为空           |
| list     | Array   | 配置的防盗链名单                                  |

#### Key防盗链参数
| 参数名称 | 类型   | 说明                    |
| -------- | ------ | ----------------------- |
| switch   | string | 开关，on:开启，off:关闭 |
| key      | string | 鉴权key                 |
