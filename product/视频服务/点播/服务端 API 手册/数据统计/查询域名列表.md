## 接口名称
DescribeVodHosts

## 功能说明

用于查询点播所有域名的信息。

## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟

### 参数说明
| 参数名称      | 必填 | 类型 | 说明                                                                            |
| ------------- | ---- | ---- | ------------------------------------------------------------------------------- |
| COMMON_PARAMS | 是   |      | 参见[公共参数](/document/product/266/7782#.E5.85.AC.E5.85.B1.E5.8F.82.E6.95.B0) |

### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=DescribeVodHosts
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称            | 类型    | 说明                                                                                                                                                                                |
| ------------------- | ------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| code                | Integer | 错误码, 0：成功；其他值：失败                                                                                                                                                       |
| message             | String  | 错误信息                                                                                                                                                                            |
| data                | Array   | 域名信息列表                                                                                                                                                                        |
| data.host           | String  | 域名                                                                                                                                                                                |
| data.cname          | String  | 点播 CDN 分配的 .cdn.dnsv1.com 后缀加速域名                                                                                                                                         |
| data.oversea        | Integer | 是否开通了海外加速，1：是，0：否                                                                                                                                                    |
| data.status         | Integer | 域名加速状态<ul><li>1：域名审核中；</li><li>2：域名审核未通过；</li><li>3：域名审核通过处于部署中；</li><li>4：域名部署中；</li><li>5：域名已启动；</li><li>6：域名已关闭 </li><ul> |
| data.sslType        | Integer | 是否开通 HTTPS，0：未开通 HTTPS 配置；若为其他，表示已开通 HTTPS 配置                                                                                                               |
| data.sslDeployTime  | Integer | HTTPS 部署时间，Unix 时间戳                                                                                                                                                         |
| data.sslExpireTtime | Integer | HTTPS 证书过期时间，Unix 时间戳                                                                                                                                                     |

### 错误码说明
| 错误码    | 含义说明                                     |
| --------- | -------------------------------------------- |
| 4000-7000 | 参见[公共错误码](/document/product/266/7783) |
| 1000      | 无效参数                                     |
