## 接口名称
DescribeCdnDetailStat

## 功能说明
用于查询指定域名在某一时间区间的流量、带宽、请求数等 CDN 统计数据明细。

### 详细说明
1. 可以查询最近90天的数据。
2. 流量的单位是 byte （字节），带宽的单位是 bps （比特每秒）。

## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟      

### 参数说明
| 参数名称      | 必填 | 类型    | 说明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ------------- | ---- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| hosts         | 否   | Array   | 域名列表，一次最多查询20个域名。如果为空，将查询用户所有点播域名的统计数据（如果域名数超过20个，则返回错误）                                                                                                                                                                                                                                                                                                                                                                                                                          |
| startDate     | 是   | String  | 起始日期，格式为 yyyy-MM-dd，如2018-03-01                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| endDate       | 是   | String  | 结束日期，格式为 yyyy-MM-dd，如2018-03-02                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| statType      | 是   | String  | CDN 统计数据类型 <ul><li> flux：表示流量明细，单位是字节（byte）；</li><li>bandwidth：表示峰值带宽明细，单位是比特每秒（bps）;</li><li>requests：表示请求数明细；</li><li>hitrate：表示平均请求命中率明细，单位为万分比</li></ul>                                                                                                                                                                                                                                                                                                     |
| period        | 否   | Integer | 明细数据时间粒度，单位为分钟<ul><li>如果查询的起始日期和结束日期跨度为一天，可以支持的时间粒度有：5分钟、15分钟、30分钟、60分钟、120分钟、240分钟、1440分钟(一天) ；</li><li> 如果时间跨度为2到3天，支持的粒度为15分钟、30分钟、60分钟、120分钟、240分钟、1440分钟；</li><li>如果时间跨度为4到7天，支持的粒度为30分钟、60分钟、120分钟、240分钟、1440分钟；</li><li>如果时间跨度为8到30天，支持的粒度为120分钟、240分钟、1440分钟；如果时间跨度大于30天，支持的粒度为240分钟、1440分钟</li></ul> 默认按时间跨度支持的最小粒度返回数据 |
| COMMON_PARAMS | 是   |         | 参见[公共参数](/document/product/266/7782#.E5.85.AC.E5.85.B1.E5.8F.82.E6.95.B0)                                                                                                                                                                                                                                                                                                                                                                                                                                                       |

### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=DescribeCdnDetailStat
&hosts.0=123.vod2.myqcloud.com
&startDate=2018-03-01
&endDate=2018-03-06
&statType=flux
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称      | 类型    | 说明                                                                                                                                        |
| ------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| code          | Integer | 错误码, 0：成功；其他值：失败                                                                                                               |
| message       | String  | 错误信息                                                                                                                                    |
| data          | Object  | 结果数据                                                                                                                                    |
| data.period   | Integer | 时间粒度，单位为分钟。如 period=5，表示明细数据是5分钟一个统计点的数据，对于 flux 是这5分钟内的总流量，对于 bandwidth 是这5分钟内的峰值带宽 |
| data.statType | String  | CDN 统计数据类型， 和输入的 statType 一致                                                                                                   |
| data.list     | Array   | 域名的统计数据列表，见 HostStatData 说明                                                                                                    |

#### HostStatData
| 参数名称     | 类型   | 说明                                                                                              |
| ------------ | ------ | ------------------------------------------------------------------------------------------------- |
| host         | String | 域名                                                                                              |
| domesticData | Array  | 国内 CDN 节点的统计数据明细，见统计数据明细 StatData                                              |
| overseaData  | Array  | 海外 CDN 节点的统计数据明细，见统计数据明细 StatData。如果域名没有开通海外 CDN 加速，忽略这个参数 |

#### StatData
| 参数名称  | 类型    | 说明           |
| --------- | ------- | -------------- |
| timeStamp | Integer | Unix 时间戳    |
| value     | Integer | CDN 统计项数值 |


### 错误码说明
| 错误码    | 含义说明                                     |
| --------- | -------------------------------------------- |
| 4000-7000 | 参见[公共错误码](/document/product/266/7783) |
| 1000      | 无效参数                                     |
