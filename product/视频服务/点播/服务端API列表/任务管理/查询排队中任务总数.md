## 接口名称
GetTaskCountInQueue

## 功能说明
该接口用于获取一个用户处于排队中的任务总数

## 请求方式

### 请求域名
vod.api.qcloud.com

### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| COMMON_PARAMS | 是 |  | 参见[公共参数](/document/product/266/7782#.E5.85.AC.E5.85.B1.E5.8F.82.E6.95.B0) |

### 请求示例

```
https://vod.api.qcloud.com/v2/index.php?Action=GetTaskCountInQueue
&COMMON_PARAMS
```

## 接口应答

### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Integer | 错误码, 0: 成功；其他值: 失败 |
| message | String | 错误信息 |
| num | Integer | 排队中任务总数 |

### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000-7000 | 参见公共错误码。  |

### 应答示例
```
{
    "code": 0,
    "message": "",
    "num": 5000
}
```
