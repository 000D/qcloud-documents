## 接口名称
GetTaskList

## 功能说明
1. 查询任务列表。

## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟

### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| vodTaskId | 是 | String | 任务流ID |
| order       | 否    | String | 结果排序，asc：按时间升序 desc：按时间降序。 不填默认为asc             |
| pageNo        | 否    | Integer | 分页页号。 不填默认为1                                     |
| pageSize      | 否    | Integer | 分页大小，范围在10-100之间。 不填默认为10                         |
| COMMON_PARAMS | 是 |  | 参见[公共参数](/document/product/266/7782#.E5.85.AC.E5.85.B1.E5.8F.82.E6.95.B0) |

### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=GetTaskList
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Integer | 错误码, 0: 成功, 其他值: 失败 |
| vodTaskList | Array | 转码模板信息列表 |
| vodTaskList.vodTaskId | String | 任务ID |
| vodTaskList.type | String | 任务类型 |
| vodTaskList.status | String | 任务状态。 如果type为procedure, 取值 PROCESSING/FINISH 之一， 否则，取值 PROCESSING/SUCCESS/FAIL 之一 |
| vodTaskList.createTime | Integer | 模板创建时间（Unix时间戳） |
| vodTaskList.updateTime | Integer | 模板信息最近更新时间（Unix时间戳） |
                 |

### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000-7000 | 参见[公共错误码](/document/product/266/7783)  |
| 1000 | 无效参数  |
| 10702 | 内部错误  |
