
在视频管理界面单击视频，屏幕右侧即可弹出视频基本信息栏，点击【视频发布】可以选择发布视频源文件 URL 地址，或视频带 Web 播放器代码。

其中，视频源文件 URL 地址对应特定码率的视频文件，本身不包含任何播放器信息。可在浏览器中直接打开，应用浏览器或操作系统自带播放器进行播放。操作方法为：

直接选择对应清晰度或码率的文件，选择预览播放或显示 URL。URL 支持直接复制或单击按钮复制，如下图所示：

![](https://mc.qcloudimg.com/static/img/785954e36b3f401a5535221fc13e24ee/image.png)

视频带 Web 播放器代码对应可用于 Web 编辑的代码，包括自适应 HTML 代码、FLASH、IFRAME。包含播放器配置信息。步骤为：

1. 点击播放器管理界面的【编辑】按钮。
![](https://mc.qcloudimg.com/static/img/783e2efebad5d7cadbc2dd74df25a548/image.png)

2. 在下拉菜单中选择已有播放器。

3. 调整播放器大小（可自定义）。
![](https://mc.qcloudimg.com/static/img/ec63a00a637524a6c92ce2ee6d1b20f6/image.png)

4. 选择是否自动播放。

5. 设置完成后点击【保存】，代码输出框即提供了对应的自适应 HTML 代码、IFRAME 信息。
![](https://mc.qcloudimg.com/static/img/1338bc5296ef497d8fca9bb9f0bd5b12/image.png)
