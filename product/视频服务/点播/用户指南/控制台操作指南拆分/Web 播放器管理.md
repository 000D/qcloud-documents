
通过 Web 播放器管理功能，您可以使用腾讯云提供的视频播放器功能，轻松定义播放器样式，随后在自己的网页中引用播放器代码以展现这些自定义样式。

具体操作方法如下：

在腾讯云控制台左侧导航栏中，点击【Web 播放器管理】。

您可以看到一个播放器列表，该列表用于管理所有的播放器。您可以自己建立并维护最多 10 个自定义播放器。其中有一个播放器（初始播放器）被设置为默认播放器。所有的视频文件在发布时，如果没有特别指明，将在 Web 播放器代码窗口应用默认播放器的设置生成播放代码。

点击【新增播放器】可以设置新增播放器的基本信息（包括名称、描述、默认画质），同时可以设计新增播放器的外观，如下图所示：

![](https://mc.qcloudimg.com/static/img/8cb78eb83a1906bbc2d6e2cd80612919/image.png)

在 Web 播放器管理界面对所有播放器进行如下操作：

预览：打开预览视频文件并查看播放器的效果。
修改：进入播放器配置状态，修改播放器的名称、画质、外观等属性。
复制：对所选择的播放器进行复制，以进一步编辑使用。
设为默认播放器：将选定播放器设置为默认播放器；
删除：删除选定播放器。如果当前播放器被视频文件所应用，则删除后，所关联的视频文件将应用默认播放器生成发布代码，该过程自动生效，无需修改代码。
>**注意：**
>默认播放器无法被删除。
