基础事件通知包括 PeerConnection 连接通知的通知。详细描述如下：

### onPeerConnectionAdd
PeerConnection 连接通知。

#### 语法示例
```javascript
    RTC.on( 'onPeerConnectionAdd' , function( info ){ })
```

#### info

| 参数        | 类型     | 描述           |
| --------- | ------ | ------------ |
| userId | String | 连接所属用户用户名 |

#### 代码示例
```javascript
    RTC.on( 'onPeerConnectionAdd' , function( info ){
        //由业务决定，是否要建立peerconnection
        if( info.userId === '指定用户名'){
            WebRTCAPI.startRTC({ userId: info.userId );
        }else{
            console.debug('不建立连接')
        }
    })
```
