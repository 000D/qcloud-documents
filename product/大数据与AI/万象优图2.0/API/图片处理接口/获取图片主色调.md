## 简介
获取图片主色调。
## 接口形式
download_url?imageInfo
## 参数说明
| 参数                                      | 含义                                       |
| --------------------------------------- | ---------------------------------------- |
|download_url                            |文档的访问链接，具体构成为&lt;bucket id&gt;-&lt;appid&gt;.&lt;picture region&gt;.&lt;domain&gt;.com/<picture  |
## 示例
```
http://examples-1251000004.picsh.myqcloud.com/sample.jpeg?imageAve
```
