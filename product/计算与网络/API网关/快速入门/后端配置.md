## 对接 mock ##
Mock 会对 API 请求返回固定配置的响应。Mock 常用于测试开发，可以在后端服务还未完成的情况下预先完成 API 配置和响应。对接 mock 时，只需要配置您的返回数据，单击【完成】即可。
![mock](//mc.qcloudimg.com/static/img/59d198b75bc21d7af480656cf6ebcc62/image.png)
