API 网关是用于实现完整 API 托管的服务，用于协助开发者轻松完成 API 的创建、维护、发布、监控等整个生命周期的管理。通过 API 网关，您可以封装后端各种服务，以 API 的形式，提供给各方使用。同时，API 网关协助您完成 API 文档管理、测试、SDK 生成。

![总体架构](https://i.imgur.com/A7jz82e.png)

无论是腾讯云的合作伙伴，还是腾讯云的客户希望能够内部使用 API 网关服务，或者有供应商希望能通过云市场将自己的能力通过 API 网关开放出去，都可以将自己的业务部署在 API 网关上，以供最终用户进行调用，最终用户无论是移动客户端，web 客户端，物联网或其他应用，都可以直接通过域名调用 API 网关中提供的 API 服务。
