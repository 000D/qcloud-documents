

### 使用弹性网卡有什么约束？
根据 CPU 和内存配置不同，云服务器可以绑定的弹性网卡数和单网卡绑定内网 IP 数有较大不同，详情请参见 [使用约束](https://cloud.tencent.com/document/product/215/6513?&_ga=1.2700851.205113617.1532487250#.E4.BD.BF.E7.94.A8.E7.BA.A6.E6.9D.9F)。


### 如何绑定云服务器？
用户绑定云服务器有以下步骤：
1. 进入 [私有网络控制台](https://console.cloud.tencent.com/vpc) 。
2.  单击左侧目录中的【弹性网卡】，进入弹性网卡列表页。
3. 找到需要绑定和配置的弹性网卡所在行，单击操作栏中的【绑定云主机】。
4. 选择需要绑定的云主机，单击【确定】。

详情请参见 [绑定云服务器](https://cloud.tencent.com/document/product/576/18535#.E7.BB.91.E5.AE.9A.E4.BA.91.E4.B8.BB.E6.9C.BA)。

### 如何配置云服务器？
弹性网卡在控制台绑定后，需要在云服务器内配置后才可正常使用，详情请参见 [配置弹性网卡](https://cloud.tencent.com/document/product/576/18535#.E9.85.8D.E7.BD.AE.E4.BA.91.E4.B8.BB.E6.9C.BA)。

### 如何分配内网 IP？
云服务器弹性网卡分配内网 IP 主要分为以下几个步骤：

1. 打开 [私有网络控制台](https://console.cloud.tencent.com/vpc)  。
2. 点击左侧目录中的【弹性网卡】，进入弹性网卡列表页。
3. 点击弹性网卡的【实例 ID】，进入弹性网卡详情页。
![](https://main.qcloudimg.com/raw/028702aac3f587be778dc142ccb3723a.png)
4. 点击【IP 管理】，查看弹性网卡上已绑定的内网 IP 和弹性公网 IP。
![](https://main.qcloudimg.com/raw/2126e5ef73cfd2fba09f444908a51cfe.png)
5. 点击【分配内网 IP】，弹窗展示 “分配内网 IP” 窗口。
![](
https://main.qcloudimg.com/raw/8558c454e20679efab3e3d6874b6cb21.jpg)
6. 选择 “自动分配” 或 “手动填写” 内网 IP。
7. 点击【新增】，在 “分配内网 IP” 窗口为弹性网卡分配多个 IP。
8. 点击【完成】，完成控制台分配内网 IP 操作。
![](https://main.qcloudimg.com/raw/df6e8723866f977bc9f0db9aee33148d.jpg)



>**注意：**
>内网 IP 还需在云服务器内配置后才正式生效。

### 如何释放内网 IP？
用户释放内网 IP 有以下步骤：
1. 进入 [私有网络控制台](https://console.cloud.tencent.com/vpc) 。
2. 单击左侧目录中的【弹性网卡】，进入弹性网卡列表页。
3. 单击需要查看的实例 ID，进入详情页。
4. 单击选项卡中的【IP 管理】，查看已绑定的内网 IP 和弹性公网 IP。
5. 单击需要释放的内网 IP 所在行的【释放】。
6. 单击【确认】。

详情请参见 [释放内网 IP](https://cloud.tencent.com/document/product/576/18538)。

### 如何绑定弹性公网 IP?
用户绑定弹性公网 IP 有以下步骤：
1. 进入 [私有网络控制台](https://console.cloud.tencent.com/vpc) 。
2. 单击左侧目录中的【弹性网卡】，进入弹性网卡列表页。
3. 单击需要绑定的实例 ID，进入详情页。
4. 单击选项卡中的【IP 管理】，查看已绑定的内网 IP。
5. 单击需绑定的内网 IP 所在行 “已绑定弹性公网 IP” 列的【绑定】。
6. 在弹出框汇中您可根据需求选择【使用现有 EIP 绑定】或【新建 EIP 绑定】。
7. 单击【确认】。

详情请参见 [绑定弹性公网 IP](https://cloud.tencent.com/document/product/576/18539)。

### 如何修改主内网 IP？
用户修改主内网 IP 有以下步骤：
1. 进入 [私有网络控制台](https://console.cloud.tencent.com/vpc) 。
2. 单击左侧目录中的【弹性网卡】，进入弹性网卡列表页。
3. 单击需要绑定的实例 ID，进入详情页。
4. 单击选项卡中的【IP 管理】，查看已绑定的主内网 IP。
5. 单击需要修改的主内网 IP 所在行的【修改主 IP】。
6. 在弹窗内输入新的主内网 IP，单击【确认】。

详情请参见 [修改主内网 IP](https://cloud.tencent.com/document/product/576/18541)。

### 如何修改弹性网卡所属子网？
用户修改弹性网卡所属子网有以下步骤：
1. 进入 [私有网络控制台](https://console.cloud.tencent.com/vpc) 。
2. 单击左侧目录中的【弹性网卡】，进入弹性网卡列表页。
3. 单击需要绑定的实例 ID，进入详情页。
4. 单击所属子网后的【更换子网】。
5. 在弹出框中选择需要更换的子网，并指定新的主 IP。
6. 单击【确定】。

详情请参见 [修改弹性网卡所属子网](https://cloud.tencent.com/document/product/576/18542)。
