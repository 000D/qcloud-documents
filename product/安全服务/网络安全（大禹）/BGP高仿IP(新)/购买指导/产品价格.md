## 转发流量
转发流量是指在腾讯云 BGP 高防机房完成清洗后转发回客户源站机房，以及从客户源站转发至客户的用户的正常业务流量。
目前有两种收费模式：
- 一种按 0.8 元 / GB 收费，按天结算。
- 一种按带宽预付费，具体价格如下：

| 带宽      | 价格（元 / 月） |

| 50 Mbps  | 4500   |
| 100 Mbps | 9000   |
| 150 Mbps | 13500  |
| 200 Mbps | 18000  |
| 500 Mbps | 45000  |
| 1 Gbps   | 90000  |
| 2 Gbps   | 180000 |
