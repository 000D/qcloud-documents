TGit 是腾讯云推出的一款开发者工具，提供基于 Git 的在线代码托管服务，包含对代码的提交、存储、下载、复刻、分支、比对、合并和历史管理；以及对项目、成员的管理等功能，大大提升研发效率。TGit具有以下特性：
#### 灵活的项目管理
- 用户通过简单的 Web 操作部署项目版本库并完成项目授权进而组建项目核心团队。
- 用户通过观看项目，可快速找到自己感兴趣的项目。
- 提供基于项目的 WIKI（百科）管理，供用户发布词条。

TGit 源于腾讯内部的代码托管平台，全面支撑腾讯各大业务，用户可享受与腾讯内部一致的服务体验。
#### 协同开发
代码存储在云端，多角色可跨地域协同开发，提高开发效率。
#### 提供代码审查
支持用户之间相互代码审查，提出对他人代码的不同看法、意见和建议，帮助提高代码质量和找出既能解决问题又能简化代码的最佳方法。
#### 安全的代码管理
- 提供细到分支级别的权限控制，重要的操作添加操作日志以方便回溯，防止重要分支上的误操作造成的开发问题。
- 支持 https 和 SSH 加密传输，防止代码泄露造成的一切问题。
