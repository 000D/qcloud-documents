### 本地仓库 JAR 包安装

目前 Spring Cloud for TSF 的第三方包暂时没有发布到公有 Maven 仓库，需要将相关 TSF JAR 包安装到本地仓库。

解压后的 SDK 下载包中提供了 Windows 和 Linux 系统下的安装脚本。

- 在 Windows 系统下，鼠标双击 `tsf_install_mvn.bat` 文件将依赖库安装到本地 Maven 仓库中。

- 在 Linux 、 macOS 系统下，在命令行下执行 `sh tsf_install_mvn.sh` 将依赖库安装到本地 Maven 仓库中。
