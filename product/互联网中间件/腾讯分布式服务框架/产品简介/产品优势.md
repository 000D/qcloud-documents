## 产品优势

| 功能项 | TSF 服务治理平台 | 自建服务治理平台 |
|--------|---------|---------|
| 服务注册中心 | 平台提供高可用的注册中心集群 | 自行搭建和维护 |
| 调用链能力 | 符合国人习惯的交互方式，与日志服务联动 | 使用开源组件，英文界面|
| 应用部署 | 成熟、灵活的部署方案 | 自行开发部署模块 |
| 日志服务 | 提供从采集、呈现、解析、搜索一站式能力 | 基于ELK等开源组件搭建 |
| 配置管理 | 分布式配置管理，从环境、版本、应用三个维度进行管理 | 自行搭建和维护 |
| 分布式事务 | 提供分布式事务解决方案，经腾讯内部多产品实践验证 | 使用开源或自行开发 |
| 微服务 API 网关 | 腾讯云 API 网关提供微服务网关能力  | 基于 Zuul 等组件开发  |
| 与腾讯云服务整合 | 深度整合 | 基于云 API 开发 |
| 售后服务 | 提供稳定及时的售后服务，强大的技术、运维团队支持 | 企业运维团队支持 |
