## 使用 CloudAudit  
腾讯云 CloudAudit 提供基于 Web 的用户界面，即控制台，如果您已注册腾讯云账户，您可以直接 登录 CloudAudit 控制台，对您的 CloudAudit 进行操作。

腾讯云 CloudAudit 提供 API 接口，这些请求属于 HTTP 或 HTTPS 请求，有关 CloudAudit API 操作的更多信息，请参阅 API 文档。

如果您倾向于使用 API 的方式对您的资源、应用和数据进行管理操作，您可以使用 SDK（支持 PHP/Python/Java/.NET/Node.js）编程或使用腾讯云命令行工具调用 CloudAudit API。

  
## 使用步骤  
1. 如果您还没完成腾讯云账户的注册，请先注册；如果您已注册腾讯云账户，直接登录到腾讯云的控制平台，在产品-管理工具-云审计这个目录下找到云审计控制平台。或者直接通过 API 登录到云审计控制平台。
2. 在云审计控制平台上有两个功能项：操作记录和跟踪集。
	 - 操作记录
	 您可以在操作记录中通过用户名，事件名称查看操作事件，这里显示只有最近7天的操作日志。
	 - 跟踪集
	 您可以新建一个跟踪集来记录您在腾讯云控制平台上的所有操作记录，创建过程中需要填写一些基本信息（跟踪名称和跟踪所有区域）、管理事件、数据事件以及存储位置。您还可以对跟踪集进行查看，编辑和删除等操作。
