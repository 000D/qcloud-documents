## COS 简介

对象存储服务（Cloud Object Storage，简称：COS）是腾讯云提供的一种存储海量文件的分布式存储服务，用户可以在 Web 上的任何位置随时存储和检索数据。腾讯云 COS 使所有用户都能使用具备高扩展性、低成本、可靠和安全的数据存储服务。

对象存储服务 COS 解决了传统文件系统访问目录资源消耗过大问题，通过 API、SDK 等多样化方式简单、快速地接入，实现了海量数据存储。通过 COS，您可以上传、下载和管理任何类型的文件。腾讯云提供了直观的 Web 管理界面，同时可以通过遍布全国范围的 CDN 节点对文件下载进行加速。
COS 当前主要提供标准存储服务，适用于数据频繁访问的场景。

### 标准存储
标准存储为用户提供了高可靠性，高可用性和高性能的对象存储服务。标准存储拥有低访问时延和较高的吞吐量，因而适用于有大量热点文件，需要频繁访问数据的业务场景。

**适用场景**：热点视频、社交图片、移动应用、游戏程序、动态网站
