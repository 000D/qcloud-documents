## 功能说明
使用 COS Browser 工具，用户可通过可视化界面，方便地进行数据的上传、下载等操作。
## 使用限制
适用于 COS V4、V5 版本

## 使用环境
### 系统环境
Windows 32/64 位系统均可
### 软件依赖
下载解压即可使用

## 下载与安装
下载后解压文件，双击 cosbrowser.exe 即可运行。
下载链接：
- [32 位版本](http://cosbrowser-1251902136.cos.ap-guangzhou.myqcloud.com/cosbrowser-1.0.2-win-32.zip) 
- [64 位版本](http://cosbrowser-1251902136.cos.ap-guangzhou.myqcloud.com/cosbrowser-1.0.2-win-64.zip)

## 使用方法
用户可使用云 API 密钥 SecretId 和 SecretKey 进行登录，该密钥可在 [访问管理控制台](https://console.cloud.tencent.com/cam/capi) 获取，登录后会保留登录信息。
目前支持批量上传、下载、删除功能。拖拽上传下载暂不支持。
