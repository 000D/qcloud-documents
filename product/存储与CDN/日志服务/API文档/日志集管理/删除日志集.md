## 删除日志集

**描述**

删除日志集

**请求行**

```
DELETE /logset
```

**请求示例**

```
DELETE /logset?logset_id=xxxx-xx-xx-xx-xxxxxxxx HTTP/1.1
Host: <Region>.cls.myqcloud.com
Authorization: <AuthorizationString>

```

**请求头**

除公共头部外，无特殊请求头部。

**请求参数**

| 字段名        |  类型  | 位置  | 是否必须 |      含义                       |
|--------------|--------|------|---------|--------------------------------|
| logset_id    | string | query| 是      |要删除的日志集的 id                |


**返回示例**

```
HTTP/1.1 200 OK
Content-Length: 0

```

**响应头**

除公共头部外，无特殊请求头部。

**返回内容说明**

无

**错误码**

见错误码说明
