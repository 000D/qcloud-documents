## 部署方式

新的 Wafer 支持 2 种部署方式。

| 方式                                       | 成本                                 | 说明                                       |
| :--------------------------------------- | ---------------------------------- | ---------------------------------------- |
| [自行部署](https://github.com/tencentyun/wafer2-startup/wiki/%E8%87%AA%E8%A1%8C%E9%83%A8%E7%BD%B2) | 时间 ★★★★★<br/>难度 ★★★★★<br/>花费 ☆☆☆☆☆ | 自行搭建所有的服务器、数据库以及所依赖<br/>的运行环境，要求开发者有丰富的运维经验。 |
| [一站式部署](https://github.com/tencentyun/wafer2-startup/blob/master/README.md) | 时间 ★☆☆☆☆<br/>难度 ★☆☆☆☆<br/>花费 ★★☆☆☆ | 由腾讯云提供的一站式解决方案，免费提供<br/>开发和生产两套环境，配合微信开发者工具<br/>一键上传、部署、测试和发布。 |

## Wafer 解决方案架构

腾讯云小程序一站式解决方案提供了从域名购买、服务器购买、证书部署到数据库部署、套件加速开发等整套小程序解决方案。

### 客户端 SDK

Wafer 客户端 SDK 封装了微信小程序客户端 API。和服务端 SDK 配套提供会话、信道、上传图片、客服消息等服务。

开发者资源：

- [客户端 SDK](https://github.com/tencentyun/wafer2-client-sdk)
- [客户端 Demo](https://github.com/tencentyun/wafer2-startup)

### 服务端 SDK

小程序业务服务器通过 Wafer 服务器 SDK 支持会话、信道、上传图片、客服消息等服务。

开发者资源：

- [Node SDK](https://github.com/tencentyun/wafer2-node-sdk)
- [Node Demo](https://github.com/tencentyun/wafer2-startup)

### WebSocket 信道 PaaS 服务

WebSocket 信道 PaaS 由腾讯云及腾讯 QQ 空间团队联合提供，提供与小程序间稳定、可靠、高效的 WebSocket 长连接。由业务服务器通过 SDK 进行服务申请，建立连接后业务服务器、信道服务和小程序之间会形成三方通讯。
