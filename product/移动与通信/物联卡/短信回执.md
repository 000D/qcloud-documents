# 短信回执
## 接口描述
### 功能描述
短信下发给物联卡之后，运营商会推送回执，用以标识下发的最终状态。此协议是给用户后台服务推送短信状态回执，需要由用户提供接收回执的地址，如果没有配置请您联系腾讯云物联卡技术支持(QQ：3513545165)。

## 请求参数
	{
	    "type": "report",
	    "iccid": "898602b8011730558259",
	    "user_receive_time": "2017-09-17 08:03:04", 
	    "report_status": "SUCCESS",
	    "sid": xxxxx
	}
|参数名称|描述|类型|必选|
|-----|-----|-----|-----|
|type|请求类型|字符串|是|
|iccid|物联卡 iccid|字符串|是|
|user_receive_time|短信送达时间|字符串|是|
|report_status|短信下发状态|字符串|是|
|sid|短信下发时返回的发送 ID|字符串|是|

## 应答参数
	{
	    "code": 0,
	    "message": "OK"
	}
|参数名称|描述|类型|必选|
|-----|-----|-----|-----|
|code|请求返回码，0 为成功，非 0 为失败|整数|是|
|message|请求返回信息|字符串|是|

**注：**
请用户接口严格按照上述格式进行应答，否则我们将进行多次（最多 3 次）推送。
