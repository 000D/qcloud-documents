### 页面排行-当天实时列表
1. 功能说明：查询当天所有 url 的 pv\uv\vv\iv 数据；
2. URL：`http://mta.qq.com/h5/api/ctr_page/list_all_page_realtime`；
3. HTTP 请求方式：GET。

**参数**

|  | 类型 | 含义 |备注 |
|---------|---------|---------|---------|
| app_id | 整数 | 	应用 ID |应用注册时生成的 ID |
| idx | 字符串 | 指标列表 |指标列表，使用“,”间隔查询指标，详见附录的指标字典 |
| sign | 字符串 | 	验证串 |见示例生成过程 |
**可查询指标**

| 指标 | 指标解释 | 计算方式 |
|---------|---------|---------|
| pv | 页面被打开的次数 | 每个页面被打开一次，计为 1 次访问 |
| uv | 访客 1 天内去除重复访问次数 | 1 天内相同访客访问次数只计 1 次访问 |
| vv | 访客 1 天内访问次数 | 	从访客来到您网站到最终关闭网站的所有页面离开，计为 1 次访问 |
| iv | 1 天内访问的 IP 数 | <br\> |

### 性能监控
1. 功能说明：按天查询对应省市的访问延时与解析时长；
2. URL：`http://mta.qq.com/h5/api/ctr_page_speed`；
3. HTTP 请求方式：GET

**参数**

|  | 类型 | 含义 |备注 |
|---------|---------|---------|---------|
| app_id | 整数 | 	应用 ID |应用注册时生成的 ID |
| start_date | 字符串 | 	开始时间 |开始时间(Y-m-d) |
| end_date | 字符串 | 	结束时间 |结束时间(Y-m-d) |
| type_contents | 字符串 | 省市 ID/运营商 ID/页面地址 |省市可选值详见附录（省市），运营商可选值详见附录（运营商），页面 url 需要 url 编码，使用“,”间隔查询|
| type | 字符串 | 	类别 |可选值详见附录（性能监控）|
| idx | 字符串 | 	指标列表 |可选值详见附录（指标字典），使用“,”间隔查询指标 |
| sign | 字符串 | 	验证串 |见示例生成过程 |
**可查询指标**

| 指标 | 指标解释 | 计算方式 |
|---------|---------|---------|
| visitor_speed | 访问延时 |用户访问总时长的的平均值|
| dns_speed | DNS 解析时长|域名服务器解析时间 |
| tcp_speed | TCP 链接时长 |TCP 与服务器端建立连接花费时长 |
| request_speed |首字节时长 |接收到服务器响应再减去发出 HTTP 请求的时间|
| resource_speed |资源下载时长 |页面所有 onload 计算范围内的资源加载结束后到文档开始进行 dom 对象解析的时间|
| dom_speed |	页面渲染时长 |页面开始构建元素到全部展示的过程所消耗的时间|
