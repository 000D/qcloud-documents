```

```
统计方法
QPS 为 3 台 cvm 机器压测的 QPS 之和。

### 性能数据
QPS 为 3 台 cvm 机器压测的 QPS 之和。

|腾讯云Redis实例规格|	连接数	|QPS值|
|---------|---------|---------|
|主从版Redis 8G	|9000	|85000|
|新一代Redis 64G	|9000	|120000|

由于集群版 Redis 由于有限流(12MB/S)的控制，QPS 最大值为：限流值/写入或者读取的 key 的 value 大小。如
redis-benchmark -h ip -a passwd -r 100000 -d 128 -t set -n 10000000 -c 100 -e 情况下，单个分片 QPS 可达 95000
在 key 分布均匀的情况下，性能随节点个数增涨而增涨。
