1) Log in to [Tencent Cloud Console][1], click "Relational database" in the navigation bar to enter [Cloud Database Console][2], and click "New" button.
2. Select an appropriate billing mode. Two modes are available: annual or monthly plan and pay-by-usage.
	For billing process of monthly or annual prepaid method, refer to [Prepaid Billing Process][3]
	For billing process of postpaid method, refer to [Unified Billing Process][4]
3. Select a region and an availability zone.
4. Select a network environment to support basic network and VPC.
5. Select instance specifications and the required disk.
6. Select a data copy method. For differences among copy methods of database instance, refer to [Database Instance Copy][5].
7. Select the project to which the database instance belongs, and the default is the default project.
8. Select quantity and purchased usage period and confirm them. Then click "Buy Now".
